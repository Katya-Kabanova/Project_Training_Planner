Планировщик тренировок
Автор: Екатерина Кабанова
Описание:
Это программа, которая помогает мне планировать мои тренировки. В ней я могу добавлять тренировки с датой, типом и временем. Также я могу искать тренировки по дате или типу, чтобы не забывать, что делал.
Примеры использования
1. Запускаете программу.
2. Вводите дату (например, 15.09.2024), тип тренировки (например, бег или силовая) и сколько минут занимала тренировка.
3. Нажимаете кнопку «Добавить тренировку», чтобы сохранить её.
4. Можно фильтровать тренировки — выбрать только бег или только тренировки на определённую дату.
5. Все данные сохраняются автоматически, когда вы закрываете программу.
Замечание
Можно добавлять много тренировок и смотреть их по разным фильтрам.
