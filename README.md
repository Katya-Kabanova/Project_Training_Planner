# Project_Training_Planner
Автор:   Кабанова Катя
## Описание
Это программа, которая помогает мне планировать мои тренировки. В ней я могу добавлять тренировки с датой, типом и временем. Также я могу искать тренировки по дате или типу, чтобы не забывать, что именно и когда делала.
## Примеры использования 
1. Запускаете программу.
2. Вводите дату (например, 15.04.2026), тип тренировки (например, бег или силовая) и сколько минут занимала тренировка.
3. Нажимаете кнопку «Добавить тренировку», чтобы сохранить её.
4. Можно фильтровать тренировки — выбрать только бег или только тренировки на определённую дату.
5. Все данные сохраняются автоматически, когда вы закрываете программу.
6. Можно добавлять много тренировок и смотреть их по разным фильтрам.
